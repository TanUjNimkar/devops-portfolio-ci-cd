# DevOps Portfolio CI/CD Project

This project demonstrates an end-to-end DevOps workflow by deploying a static website using Docker, CI/CD, and AWS.

## 🚀 Tech Stack
- Docker
- Nginx
- GitHub Actions (CI/CD)
- AWS EC2
- Linux

## ⚙️ Features
- Containerized static website using Docker
- Automated build and deployment using GitHub Actions
- Deployment on AWS EC2
- Scalable and repeatable deployment process

## 📦 Project Workflow
1. Code pushed to GitHub
2. GitHub Actions builds Docker image
3. Image pushed to DockerHub
4. Deployed on AWS EC2
